Takuzu 02E

Rules

To win the game, each square needs to be either an orange or a watermelon following certain rules :

  • Three consecutive squares cannot be of the same fruit
  • Each row and each column should contain the same number of oranges and watermelons

Left click on a square to put or remove an orange. Right-click to put or remove a watermelon
